EagleTools
==================

Miscellaneous scripts for extending EAGLE CAD

Installation
============

Manual 
------
- Copy `scr/*.scr` file into your EAGLE scr directory
- Copy `ulp/*.ulp` file into your EAGLE ulp directory

Execute
============
```
script makepng
```
```
run getelem 
```
```
run resize 
```
```
run poly 
```
